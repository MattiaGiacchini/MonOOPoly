## Informazioni:

1. Installare una JRE funzionante.
2. Scaricare **monoopoly.jar** nella sezione download.
3. Per lanciare l'applicazione eseguire il comando 'java -jar monoopoly.jar' da terminale.
4. Iniziare a giocare!
